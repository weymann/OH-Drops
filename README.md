# openHAB Marketplace Drops

Repository storing openHAB jars / readme published on [openHAB Community Marketplace](https://community.openhab.org/c/marketplace/69)

## basic-profiles

Added new time-weighted average profile

## dirigera

Introduce new Matter devices like TIMMERFLOTTE, KAJPLATS, ...

## energycharts

Pricing and renewable energy measurements delivered by [Frauenhofer ISE](https://www.energy-charts.info/).
Work in prgress

## energyforecast

Energy price forecast beyond EPEX day-ahead pricing using ML via [Energy Forecast Service](https://www.energyforecast.de/) 

## geocoding

Transformation service to translate GPS coordinates into human readable addresses using [Nominatim](https://nominatim.org/)

## mercedesme

Enhance `charge#status` state options for actual vehicles

## sensorcommunity

Added `radiation` sensor
